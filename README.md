# Hawk

**Your Valorant companion.** Hawk reads your local Riot session to give you **live match analysis**, your **daily store**, and instant **player lookup** — in one clean desktop app.

## Download

### [⬇ Download the latest installer →](https://github.com/49ghoul/Hawk/releases/latest)

Grab the installer (**`.exe`**) from the latest release, run it, and launch **Hawk**.

- **Windows only.**
- Hawk **keeps itself up to date** — new versions install automatically.

## What you get

- **Live match analysis** — agents, ranks, and stats for everyone in your lobby the moment you load in.
- **Store** — your daily store and bundles at a glance.
- **Player lookup** — search any Riot ID for a full profile and match history.

## Notes

- Hawk reads your **local** Riot/Valorant session on your own PC — it doesn't need your password.
- It's a companion overlay/tool; it doesn't modify the game.
